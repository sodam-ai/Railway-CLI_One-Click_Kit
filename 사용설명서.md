# 왕초보 사용설명서 — Railway 원클릭 키트

컴퓨터·인터넷·AI 도구가 처음인 분도 따라 할 수 있게, **화면에 보이는 그대로** 적었습니다. 어려운 말은 괄호로 풀어 썼습니다.

> **Railway(레일웨이)** = 내가 만든 앱·서버·데이터베이스를 **인터넷에 올려 실행**해 주는 서비스입니다. (예: 챗봇, 백엔드, API)
>
> 📄 이 문서와 **같은 내용의 PDF**: `사용설명서.pdf` · 🇬🇧 영어: `GUIDE.en.md` / `GUIDE.en.pdf`
> 📘 더 자세한 종합 설명: `README.md` / `README.pdf`

---

## 0. 시작 전에 (딱 한 가지)

폴더 안의 **`시작하기.bat`** 를 **더블클릭**하세요.
파란 글씨로 **내 컴퓨터 상태**와 **"지금 할 일"** 이 한 줄로 나옵니다. 그 안내만 따라가면 됩니다.

> - 검은 창(또는 파란 창)이 떠도 정상입니다.
> - 메뉴는 **위/아래 화살표로 고르고 Enter**, 또는 **숫자키**를 눌러도 됩니다.
> - **관리자 허용 창(UAC)은 뜨지 않습니다.** 이 키트는 관리자 권한이 필요 없습니다. (혹시 다른 프로그램의 허용 창이 뜬다면 이 키트와 무관한 것입니다.)

## 0-1. 파일이 안 열리거나 "보호됨" 경고가 뜰 때 (다운로드 차단 풀기)

인터넷에서 받은 파일은 윈도우가 안전을 위해 **잠가둘 때**가 있어요. 풀어주면 됩니다.

- **방법 1 (가장 쉬움):** 압축을 풀기 **전에**, 받은 zip 파일을 **우클릭 → 속성 → 맨 아래 '차단 해제(Unblock)' 체크 → 확인**. 그다음 압축을 푸세요.
- **방법 2:** `시작하기.bat` 를 **우클릭 → 속성 → '차단 해제' 체크 → 확인** 한 뒤 더블클릭.
- 파란 **"Windows의 PC 보호"** 창이 뜨면 → **추가 정보 → 실행**.

> 한 번 `시작하기.bat` 가 실행되면, 폴더 안 나머지 파일의 잠금은 **자동으로 풀립니다.** (보통 처음 한 번만 신경 쓰면 됩니다.)

---

## 1. 설치하기 (처음 한 번만)

1. `시작하기.bat` → 화면에서 **[1] 설치 / 업데이트** 선택.
2. **검은 창**이 영어로 자동 진행됩니다. **닫지 말고 기다리세요** (보통 1~10분).
   - ✅ **관리자 허용 창은 뜨지 않습니다.** 그냥 기다리면 됩니다.
3. `INSTALLATION COMPLETE` (또는 비슷한 완료 메시지)가 보이면 성공입니다.
4. 끝나면 안내판 창으로 돌아와 **아무 키나** 누르세요. 상태를 다시 확인합니다.

**"Node.js(필수 부품)가 없다"고 나오면?**
→ 그래도 키트가 **다른 설치 방법(프로그램 파일 직접 내려받기)을 자동으로 시도**합니다. 그래도 안 되면 직접 받으세요:
1. 인터넷 브라우저에서 **https://nodejs.org** 접속
2. 큰 초록 버튼의 **LTS 버전**(숫자 20 이상 권장)을 받아 설치
3. 창을 닫고 **`시작하기.bat` 를 다시** 실행 → 그다음 **[1] 설치 / 업데이트**

> 설치 직후 "railway 를 못 찾는다"고 나오면 → **[9] 내 상태 점검 → [2] PATH 새로고침** 을 누르거나, 창을 모두 닫고 **새 창**에서 `시작하기.bat` 를 다시 여세요.

---

## 2. 로그인 (Railway 계정 연결, 처음 한 번만)

1. `시작하기.bat` → **[2] 로그인** 선택.
2. 인터넷 브라우저가 저절로 열립니다 → **내 Railway 계정으로 로그인 / 권한 허용(Authorize)**.
   - 계정이 없으면 https://railway.com 에서 **무료 가입** 먼저 하세요.
3. "성공" 화면이 나오면 브라우저를 닫고 안내판으로 돌아오세요.

> - 로그인은 한 번만 하면 됩니다. 다음부터는 기억합니다.
> - 잘 됐는지 확인: **[3] 로그인 상태 확인**.
> - 내 프로젝트들 보기: **[4] 내 프로젝트 보기**.

## 3. 자주 쓰는 작업 메뉴 (배포·로그·상태)

홈에서 **[5] 자주 쓰는 작업** 을 누르면, 어려운 영어 30개 메뉴 대신 **쉬운 한국어 메뉴**가 열립니다.

| 번호 | 무엇을 하나요 |
|---|---|
| 1 | **프로젝트 시작** — 이 폴더로 새 프로젝트 만들기 |
| 2 | **프로젝트 연결** — 이미 있는 프로젝트에 연결 |
| 3 | **배포하기** — 이 폴더를 인터넷에 공개 *(YES 확인)* |
| 4 | 다시 배포 — 최근 내용을 다시 공개 *(YES 확인)* |
| 5 | 실시간 로그 보기 — 서버 기록 보기 (멈춤: Ctrl+C) |
| 6 | 프로젝트 상태 — 지금 상태 확인 |
| 7 | 환경변수 보기 — 설정값 목록 |
| 8 | 전체 30기능 열기 — 고급 메뉴(새 창) |
| 0 | 뒤로 (홈으로) |

## 4. 배포 준비 → 배포하기 (순서가 중요!)

순서: **로그인 → 프로젝트 시작/연결 → 배포**

1. 내 앱(코드) 파일이 있는 폴더에서 시작하는 것이 좋습니다.
2. `시작하기.bat` → **[5] 자주 쓰는 작업** → **[1] 프로젝트 시작** (또는 이미 있으면 **[2] 프로젝트 연결**).
   - 영어로 질문이 나오면 **방향키(↑↓)와 Enter** 로 고르면 됩니다.
3. **[5] → [3] 배포하기** 선택.
4. **"공개하려면 대문자로 YES 입력"** 이 나오면 → 키보드로 **`YES`** 입력 후 Enter. (그냥 Enter면 취소됩니다.)
5. 잘 올라갔는지 **[5] → [5] 실시간 로그 보기** 로 확인하세요.

> 배포 = 내 앱을 **Railway 서버에 올려 실행**시키는 일입니다. 인터넷에 공개되므로 **꼭 내 앱 폴더에서, 프로젝트를 연결/시작한 뒤** 하세요.

## 5. 막혔을 때 — 내 상태 점검 (자가진단)

`시작하기.bat` → **[9] 내 상태 점검 / 문제 해결**.

한 화면에서 **Node.js · Railway · 설치 위치 · 로그인 · 인터넷** 을 자동으로 확인하고, **무엇이 문제고 뭘 누르면 되는지** 알려줍니다.

아래 빠른 해결 버튼이 있습니다:
- **[1]** 설치 / 업데이트 다시 하기
- **[2]** PATH 새로고침 — *방금 깔았는데 "없음"으로 뜰 때* 즉효
- **[3]** 다시 점검
- **[0]** 뒤로

---

## 6. 제거하기 (지우기)

1. `시작하기.bat` → **[6] 제거하기**.
2. 무엇이 지워지는지 화면 안내를 보고, **대문자 `DELETE`** 를 입력해야 진행됩니다.
3. 깨끗이 지워집니다.

**안심하세요 — 제거해도 다음은 그대로 둡니다:**
- 내가 만든 **앱(코드) 파일**
- **Node.js**
- Railway 서버에 올라간 **내 앱**

> 프로그램만 지웁니다. 프로젝트 폴더의 숨김 연결정보(`.railway`)는 실수 방지를 위해 자동으로 지우지 않습니다.

## 7. 오류가 났을 때 (당황하지 마세요)

| 이런 화면이 나오면 | 이렇게 하세요 |
|---|---|
| "Windows의 PC 보호 / 차단됨" | 파일 우클릭 → 속성 → **차단 해제** 체크 → 확인. 또는 **추가 정보 → 실행** |
| `RAILWAY CLI IS NOT INSTALLED` | 먼저 `시작하기.bat` → **[1] 설치 / 업데이트** |
| 설치했는데 `railway ... not found` | **[9] → [2] PATH 새로고침**, 또는 창 닫고 새 창에서 다시 실행 |
| Node.js 관련 경고 | 그대로 두면 다른 방법 자동 시도. 안 되면 https://nodejs.org 에서 **LTS** 설치 후 다시 [1] |
| `install did not finish` 비슷한 메시지 | **[9] → 인터넷 연결** 확인. 백신을 10분 끄고 다시 설치 |
| 배포가 실패함 | 내 앱 폴더에서 **[5]→[1] 프로젝트 시작** 또는 **[2] 연결** 먼저 |
| 메뉴에서 엉뚱한 번호 입력 | 그냥 다시 메뉴로 돌아갑니다. 걱정 마세요 |
| 한국어 안내가 안 열려요 | `INSTALL.bat → RUN.bat → UNINSTALL.bat` 직접 더블클릭(영어) |

## 8. 자주 묻는 질문

**Q. 명령어를 외워야 하나요?**
A. 아니요. 화살표나 번호만 누르면 됩니다. (고급 사용자는 전체 메뉴(RUN.bat) **30. 직접 명령 입력** 에서 명령을 직접 칠 수 있습니다.)

**Q. 관리자 권한이 필요한가요?**
A. 아니요. **관리자 허용 창은 뜨지 않습니다.** 프로그램이 내 계정 폴더에만 설치됩니다.

**Q. 돈이 드나요?**
A. Railway는 사용량 기반 요금/무료 크레딧 정책이 있습니다. 사용량이 많아지면 요금이 생길 수 있으니 https://railway.com 대시보드에서 확인하세요. (이 키트 자체는 요금과 무관합니다.)

**Q. 잘못 눌러서 망가지면요?**
A. 위험한 작업(배포·서비스 내리기·제거)은 모두 **`YES` 또는 `DELETE` 를 한 번 더** 입력해야 진행됩니다. 그냥 창을 닫아도 됩니다.

**Q. 내 파일·로그·설정은 어디에 있나요?**
A. 로그인 정보 `%USERPROFILE%\.railway\config.json`, 설치 기록 `%TEMP%\railway_install.log`. (자세한 위치는 `README.md` 13장 참고.)
